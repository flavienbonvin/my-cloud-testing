WTF is that

Tailwind CSS works by scanning all of your HTML files, JavaScript components, and any other templates for class names, generating the corresponding styles and then writing them to a static CSS file.

It's fast, flexible, and reliable — with zero-runtime.

The simplest and fastest way to get up and running with Tailwind CSS from scratch is with the Tailwind CLI tool. The CLI is also available as a standalone executable if you want to use it without installing Node.js.

  • Install Tailwind CSS
    1. Install tailwindcss via npm, and create your tailwind.config.js file.
    2. Add the paths to all of your template files in your tailwind.config.js file.
  • Add the Tailwind directives to your CSS
    1. Add the @tailwind directives for each of Tailwind’s layers to your main CSS file.
    2. Run the CLI tool to scan your template files for classes and build your CSS.
  • Start the Tailwind CLI build process
  • Start using Tailwind in your HTML

Get familiar with some of the core concepts that make Tailwind CSS different from writing traditional CSS.
